# Function Driven

Solana Suite provides what you want to do on a function basis. It is easy for
beginners because it can be achieved by calling a single function.

The usage is as follows

```js
"Module Name"."Function Name"
```

#### Transfer SOL

```js
SolNative.transfer();
```

#### Mint SPL-TOKEN

```js
SplToken.mint();
```

#### Mint cNFT

```js
CompressedNft.mint();
```

#### Find NFT

```js
RegularNft.find();
```
